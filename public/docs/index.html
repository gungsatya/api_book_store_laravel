Introduction

This API for educational purpose with study case book store. Created by Satya Wibawa.

Base URL: http://localhost:8000

Authenticating requests

This API is not authenticated.

Author Management

API collection to manage authors data.

Author collections

Get a list of authors.

Example request:
curl --request GET \
    --get "http://localhost:8000/api/v1/authors?q=Eileen&pageSize=10&page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"pageSize\": 10,
    \"q\": \"Eileen\"
}"

const url = new URL(
    "http://localhost:8000/api/v1/authors"
);

const params = {
    "q": "Eileen",
    "pageSize": "10",
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "pageSize": 10,
    "q": "Eileen"
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
