Introduction

This API for educational purpose with study case book store. Created by Satya Wibawa.

Base URL: http://api_book_store_laravel.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Author Management

API collection to manage authors data.

Author collections

Get a list of authors.

Example request:
curl --request GET \
    --get "http://api_book_store_laravel.test/api/v1/authors?q=Eileen&pageSize=10&page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"pageSize\": 10,
    \"q\": \"Eileen\"
}"

const url = new URL(
    "http://api_book_store_laravel.test/api/v1/authors"
);

const params = {
    "q": "Eileen",
    "pageSize": "10",
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "pageSize": 10,
    "q": "Eileen"
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
